# Capacitor Time Notification

## ⚙️ Environment Setup
- **Node.js**: v23.9.0
- **Java**: OpenJDK 17
- **IDE**: Neovim
- **Build Tool**: CLI

## 🛠 Build Commands

### Build Web Assets
```bash
npm run build
```

###Sync with Capacitor
```bash
npx cap sync
```

### Run Local Server
```bash
npx http-server dist --port 3000 --cors -o
```

### 📦 APK Location
#### Get file apk from
```bash
cd android/app/build/outputs/apk/debug/
```
